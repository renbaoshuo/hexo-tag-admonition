## Installation

1. Install the plugin.
    ```
    npm install hexo-tag-scode --save
    ```
2. Add [styles](style.css) to your theme.

## Usage

Type this in your markdown file:

```
{% info title %}
content
{% endinfo %}

{% warn title %}
content
{% endwarn %}

{% error title %}
content
{% enderror %}
```
